# AX 전환 PM 집중과정 (4주)

경력 기획자·PM을 위한 AI 전환(AX) 실무 과정 — **1기 시범 · 선발제** 모집 소개 페이지.

- **편성**: 3시간 × 4회 = 총 12시간 · 주 1회 × 4주 · 오프라인(서울) · 10명 내외
- **지원 흐름**: 지원서 접수 → 비공개 설명회(웨비나) → 선발 → 선정자 등록·수강
- **커리큘럼**: AX 종합 전환 → 생성형 AI 실무 → AX 진단·기획 → 공공 AX 제안 실무
- **활동 영역**: 공공 AX에서 시작해 기업 AX로 확대
- **수료 연계**: 초기 핵심 프로젝트 우선 매칭 (고용보험 환급 시 최대 90%)
